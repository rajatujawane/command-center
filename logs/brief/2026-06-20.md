## Shopify

- Shopify's Spring '26 Edition shipped June 18 with the Universal Commerce Protocol — three MCP servers (Catalog, Cart, Checkout) that let AI agents browse and buy from any Shopify store; B2B checkout via UCP is still in preview but the Checkout MCP is live. (shopify.com/news/spring-26-edition-dev)
- Non-Plus stores hit a hard checkout extensibility deadline August 26 — Additional Scripts, Thank You page scripts, and legacy Order Status customizations all stop firing that day, and Shopify auto-upgrades. (flatlineagency.com/blog/shopify-checkout-extensibility-2026/)
- EU right-of-withdrawal button requirement went live June 19 — merchants selling to EU customers must surface a visible cancellation link that works without login. (community.shopify.dev)
- Shopify removed the ITEM_NOT_STOCKED_AT_LOCATION inventory API error in 2026-10 — inventory can now be adjusted at any location, so any app with error-handling logic for that code needs a cleanup pass. (shopify.dev/changelog)

💡 No app today lets B2B merchants route to different net terms by order value at checkout — native net terms are flat per company, but a Payment Customization Function could tier net-30 / net-60 / deposit-required by cart total; with UCP's Checkout MCP coming, agents will need this logic too and there's nobody building it.
